## Getting Started

Simulador de las eliminatorias CONMEBOL rumbo al mundial 2026 con Java swing/awt

Usar "/src/biblioteca/App.java" para encontrar el Main

Puede cambiar los equipos por los que desee, hay dos ejemplos, los 10 equipos de la CONMEBOL o 4 equipos simulando un grupo del mundial de acuerdo a las instrucciones comentadas en App.java

## Advertencia

El programa puede tener errores si se usa un número impar de equipos

## Dependency Management

# 1. Conectar repositorio local con GitHub
git remote add origin https://github.com/ElDani271/Simulador-Equipos-F-tbol-Java-swing-awt-.git

# 2. Subir código
git branch -M main
git push -u origin main
